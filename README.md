# CRM Web Application

## Overview

This project is a Customer Relationship Management (CRM) web application developed using Python Flask and SQLite. It allows users to manage records, authenticate access, and interact with stored data through a structured web interface.

## Features

* User authentication and login handling
* Data storage and retrieval using SQLite
* Structured backend routing with Flask
* Clean frontend interface using HTML/CSS

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS

## Learning Outcomes

* Backend web development fundamentals
* Database integration
* Debugging and logical problem solving
* Understanding full application workflow

## How to Run

1. Clone repository
2. Install dependencies
3. Run Flask server
4. Open browser to view app
